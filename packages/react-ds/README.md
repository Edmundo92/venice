## Using as lib

### Install

```sh
yarn add @juntossomosmais/venice-react
```

### Usage

```js
import { Foo } from '@juntossomosmais/venice-react'
```

And import the default theme file (or create your own theme):

```js
import '@juntossomosmais/venice-react/themes/default.css'
```
