## Contextual colors

This is the contextual colors, is important use it to easily change a theme and maintain a consistent between the projects.

This use 3 kinds of variables:

- `foo-color`: A main state
- `foo-color-lighten`: For a hover state, for example
- `foo-color-contrast`: To using inside a color, as a contrast
