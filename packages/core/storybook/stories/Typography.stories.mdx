# Typography

The typography using inside project and their CSS Variables. By default, all typography fonts use `rem` as unit to make easier change size between different viewports.

## Default values 

| Variable             | Value                 |
| -------------------- | ----------------------|
| `var(--font-family)` | 'Roboto', sans-serif' |
| `var(--text-color)`  | #4444444              |

## Sizes

<Typeset
  fontSizes={['12px', '16px', '25px', '38px']}
  sampleText="Venice is the best!"
/>

## Variants

We have variables to font and letter spacing, you just need to use on `font` and `letter-spacing` property.

<Preview>
  <h1 style={{
    font: 'var(--typography-panel-font)',
    letterSpacing: 'var(--typography-panel-letter)'
  }}>
    Panel
  </h1>
</Preview>


```css
foo {
  font: var(--typography-panel-font);
  letter-spacing: var(--typography-panel-letter);
}
```
