# Typography

The typography using inside project and their CSS Variables. By default, all typography fonts use `rem` as unit to make easier change size between different viewports.

<Typeset
  fontSizes={['12px', '16px', '25px', '38px']}
  sampleText="Venice is the best!"
/>

## Using

### Variable values

| Variable             | Value                 |
| -------------------- | ----------------------|
| `var(--font-family)` | 'Roboto', sans-serif' |
| `var(--text-color)`  | #4444444              |

### Class values

| Typography           | Value                 |
| -------------------- | ----------------------|
| Panel                | `jsm-t-panel`         |
| Title                | `jsm-t-title`         |
| Subheader            | `jsm-t-subheader`     |
| Body                 | `jsm-t-body`          |
| Body2                | `jsm-t-body2`         |
| Caption              | `jsm-t-caption`       |
| Button               | `jsm-t-button`        |

We have variables to font and letter spacing, you just need to use on `font` and `letter-spacing` property. You can just use a class to add a typography styles too.

<Preview style={{
  padding: `var(--space-xs) var(--space-sm)`
}}>
  <h1 class="jsm-t-panel">Panel</h1>
</Preview>

```html
<h1 class="jsm-t-panel">Panel</h1>
```

```css
foo {
  font: var(--typography-panel-font);
  letter-spacing: var(--typography-panel-letter);
}
```
